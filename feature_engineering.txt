1. Scaling/Normalization
StandardScaler, MinMaxScaler, RobustScaler.
Digunakan jika model sensitif terhadap skala (SVM, KNN, NN, regresi).

2. Feature Creation
Polynomial features (interaksi, pangkat).
Aggregasi (mean, sum, count pada grouped data).
Ekstraksi fitur waktu (hari, bulan, tahun, weekday).
Domain-specific features (misal rasio, selisih antar fitur).

3. Feature Selection
Univariate selection (SelectKBest, SelectPercentile).
Model-based selection (feature importance dari tree model).
Recursive Feature Elimination (RFE).

4. Dimensionality Reduction
PCA untuk kompresi fitur numerik.
t-SNE/UMAP untuk visualisasi.

5. Outlier Handling
Capping/winsorizing nilai ekstrem.
Menghapus outlier jika jelas error.

6. Interaction Features
Buat fitur baru dari hasil perkalian, pembagian, atau selisih antar fitur.

7. Log Transformation
Untuk fitur numerik yang sangat skewed.

8. Binning/Discretization
Ubah fitur numerik menjadi kategori (misal umur jadi kelompok umur).

9. Constant/Low Variance Feature Removal
Hapus fitur yang nilainya konstan atau hampir konstan.
